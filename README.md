## Weathering - an weather app built with NextJS and TypeScript

Tools used for development:
- React
- TypeScript
- Next.JS
- Tailwind

Other modules/dependencies:
- React-icons
- Dotenv
- Axios
- ESlint (from Next)
- PostCss (from Next)

APIs:
- OpenWeather
- Unsplash
- Geoapify

The deployed version on Vercel: https://weathering-two.vercel.app/
